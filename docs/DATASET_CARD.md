# FormInv Dataset Card

The **authoritative** FormInv dataset card lives on the Hugging Face Hub as the dataset
repo's `README.md`. This file is only a pointer to avoid drift.

- **Hub card (source of truth):** https://huggingface.co/datasets/forminv/forminv
- **In-repo copy:** [`hf_release/README.md`](../hf_release/README.md)

The card documents the full schema, the `all` / `canonical` / `surface` configs, per-config
row counts (primary edition `v3_103`: 760 / 103 / 657), dataset creation and provenance
(Lean 4 / Mathlib, Apache-2.0 upstream), considerations, licensing (CC-BY-4.0 data, MIT
code), and citation information.

For the Datasheets-for-Datasets questionnaire (Gebru et al.), see
[`DATASHEET.md`](DATASHEET.md).
